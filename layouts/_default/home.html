ListenUp is the self-hosted audiobook server that scans your library, fetches beautiful metadata, and streams to native apps on every device you own.

Install the server Get the apps
Quick start bash 
# Pull and run the latest server
$ docker run -d --name listenup \
    -p 8970:8970 \
    -v ./config:/config \
    -v ./audiobooks:/audiobooks \
    listenup/server:latest

# → open http://localhost:8970 to finish setup
